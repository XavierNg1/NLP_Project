 The process of calling a cellphone is highly similar to the process of calling a landline phone. You would again start by dialing the international exit code for the US, which is 011., From there, dial Australia's country code. This signals the call should be directed to Australia. The country code for Australia is "61.", The number "4" indicates you are calling a cellphone. You need to dial the number "4" if the number you're trying to reach is not a landline., From here, you can dial the phone number. As with a landline, ignore the number "0" before a phone number. This is a trunk code, which is only necessary for national calls.To review, to call a cell phone in Australia you would dial "011-61-4" followed by the cell phone number.
