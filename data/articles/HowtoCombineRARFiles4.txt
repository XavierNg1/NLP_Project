 As with PC, there are few freeware applications capable of handling RAR files, and the functionality of these products is limited. Perform an internet search using "open RAR with Mac" to produce a small list of freeware applications capable of handling RAR files.
 To set the file association, select "Settings," then "File Associations," and put a check in the box labeled ".rar." Right click the file and select "Open." The downloaded application will automatically unpack, reassemble and play the file.
