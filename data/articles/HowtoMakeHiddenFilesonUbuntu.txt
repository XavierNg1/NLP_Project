 Move the file so that it is prefixed with a dot. For example, to move a file from the command line, type: mv file .file. This will also work with directories., prefix or, if you simply want to view it from the file manager, select "Show Hidden Files." or press Ctrl-H. From the command line, type: ls -a, to list all files. To search for this file, run: ls -a | grep *pattern* for files that match "*pattern*", using standard wildcards.
